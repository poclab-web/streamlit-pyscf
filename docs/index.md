## Overview

streamlit-pyscfは、PySCFとRDKitを基盤とした、ウェブベースの計算化学ツールです。

## Features

- 🧬 **分子構造入力**: SMILES、XYZ座標での入力
- ⚡ **量子化学計算**: DFT、HF計算の実行
- 📊 **可視化**: 分子構造、軌道、スペクトルの表示
- 💾 **データ管理**: 計算結果の保存・検索

## Quick Start

```bash
pip install -r requirements.txt
streamlit run ComputationalChemistryTool.py
```
